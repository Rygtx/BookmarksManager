# 书签管理器 (Bookmarks Manager)

一个强大的跨平台书签管理工具，支持多种格式的书签导入导出，提供直观的用户界面和丰富的管理功能。

## 功能特性

### 1. 多格式支持
- HTML 格式书签文件导入
- JSON 格式书签文件导入
- 自动解析书签的标题、URL、图标和添加时间

### 2. 智能解析
- 自动识别书签文件格式
- 保持原有的文件夹层级结构
- 智能提取书签标题和URL
- 支持 favicon 图标显示

### 3. 双栏视图
- 左右两侧书签树状显示
- 支持文件夹的展开/折叠
- 支持书签编辑和删除

### 4. 差异对比
- 支持左右两侧书签的对比
- 差异统计（新增、修改、删除）
- 差异可视化展示
- 支持手动标记差异
- 使用颜色区分差异类型：
  - 绿色：新增的书签
  - 蓝色：修改的书签
  - 红色：删除的书签

### 5. 差异导航
- 差异项目导航按钮
- 差异计数显示
- 自动定位到差异项
- 差异项目高亮显示

### 6. 书签管理
- 文件夹嵌套支持
- 书签编辑功能
- 书签删除功能
- 统计信息显示

### 7. 导出功能
- 支持导出为Chrome HTML书签文件
- 支持导出为Firefox HTML书签文件
- 支持导出为Chrome JSON书签文件
- 支持导出为通用HTML格式
